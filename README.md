# [panchocorderos| Terminal](https://terminal.panchocorderos.com)

A terminal style website

## Quick Start

### Using npm/yarn

1. Install dependencies:

```bash
yarn install
```

2. Build the project:

```bash
yarn build
```

3. Run the server:

```bash
yarn start
```

## Configuration

Here's a sample of the `config.json` file:

```json
{
  "bioUrl": "https://raw.githubusercontent.com/panchocorderos/panchocorderos/master/README.md",
  "social": {
    "instagram": "panchocorderos",
  },
  "theme": "gruvboxdark" // list of themes available in themes.json
}
```
